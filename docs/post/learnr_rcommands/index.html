Learning statistics - a few useful R commands

Oct 9, 2020 · 1 min read

Some useful R commands for students learning fundamentals of statistics

1. Compute probabilities using normally distributed data

pnorm(t, mean, sd)
# pnorm(); the function pnorm returns the value of the cumulative density function (cdf) of the normal distribution given a certain random variable t, a population mean μ and population standard deviation σ.
# The syntax for using pnorm is as follows:
# pnorm(t, mean, sd)

## Example

t = 1.645
mean = 0
sd = 1

p = pnorm(t, mean, sd)

### Print the outcome

print(p)
## [1] 0.9500151

\[ pnorm(x, \mu, \sigma) = cdf(x) = \] \[ = F(x) = \displaystyle \frac{1}{\sigma \sqrt{2\pi}}\int_{-\infty}^{x} \exp\left\{ -\frac{1}{2} \left( \frac{t-\mu}{\sigma} \right)^2 \right\}\,dt \]

Z-score

In statistics, the standard score is the number of standard deviations by which the value of a raw score (i.e., an observed value or data point) is above or below the mean value of what is being observed or measured. Raw scores above the mean have positive standard scores, while those below the mean have negative standard scores. (read more: https://en.wikipedia.org/wiki/Standard_score)

\[ \text{z-score} = z(x) = \frac{x - \mu_{X}}{\sigma_{X}} \]

Cumulative Distribution Function of the Standard Normal Distribution

\[ F(x) = \displaystyle \frac{1}{\sqrt{2\pi}}\int_{-\infty}^{x} e^{\frac{-z^2}{2}}\,dz \]
