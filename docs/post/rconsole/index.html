Learning statistics using technology - useful tools and services for students and teachers

Popular and powerfull statistical software useful for learning or teaching statistics

  1. The R Project for Statistical Computing: https://www.r-project.org/
  2. RStudio IDE: https://rstudio.com/

Point-and-click graphical user interfaces (GUIs) for R programming language

  1. The jamovi project (2020). [Computer Software]. Retrieved from https://www.jamovi.org

Jamovi is built on top of the R statistical language. Jamovi offers a spreadsheet user interface; it is point-and-click rather than code-and-run.

Run R or Python in your web browser

  1. CoCalc: https://doc.cocalc.com/getting-started.html
  2. Jupyter:
  3. OneCompiler: https://onecompiler.com/r
  4. Programiz: https://www.programiz.com/python-programming/online-compiler/
  5. Repl.it: https://repl.it/
  6. Rdrr.io: https://rdrr.io/snippets/
  7. RStudio IDE Cloud: https://rstudio.cloud/

Useful online scientific calculators (mobile versions available)

  1. Geogebra:
  • https://www.geogebra.org/calculator (iOS and Android apps available)
  • https://www.geogebra.org/scientific?lang=en (iOS and Android apps available)
  1. MS Math Solver: https://math.microsoft.com/en (iOS and Android apps available)
  2. Photomath: https://photomath.app/en/ (iOS and Android apps available)

Data sets useful for teaching or learning statistics

  1. Bank danych lokalnych GUS: https://bdl.stat.gov.pl/BDL/start
  2. CORGIS: The Collection of Really Great, Interesting, Situated Datasets: https://think.cs.vt.edu/corgis/csv/
  3. DataHub.io - A collection of datasets that includes lists of countries, populations, economic data, and more: https://datahub.io/collections
  4. Datasets for teaching: https://www.sheffield.ac.uk/mash/statistics/datasets
  5. European Social Survey: https://www.europeansocialsurvey.org/
  6. Kaggle Datasets - A collection of datasets for predictive modeling and machine learning: https://www.kaggle.com/datasets
  7. OECD Data: https://data.oecd.org/
  8. Our World in Data: https://ourworldindata.org/
  9. Some datasets for teaching data science: https://simplystatistics.org/2018/01/22/the-dslabs-package-provides-datasets-for-teaching-data-science/
  10. The World Bank Data Bank: https://databank.worldbank.org/home.aspx
  11. Untidydata: https://www.jvcasillas.com/untidydata/

Create datasets and store your data in the cloud

  1. Use Google Sheets (free) or MS Excel (Office 365) to create database;
  2. Use cloud storage systems such as Dropbox, Google Drive or MS OneDrive to keep your data on the cloud.

Learning R (free online study materials)

  1. RStudio Education: https://education.rstudio.com/learn/
  2. Modern Dive. Statistical Inference via Data Science: https://moderndive.com/
  3. Navarro, D. Learning Statistics with R: https://learningstatisticswithr.com/
