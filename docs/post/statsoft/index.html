Point-and-click graphical user interfaces (GUIs) for R programming language

  1. The jamovi project (2020). [Computer Software]. Retrieved from https://www.jamovi.org

Jamovi is built on top of the R statistical language. Jamovi offers a spreadsheet user interface; it is point-and-click rather than code-and-run.

Some useful R commands for students learning fundamentals of statistics

Compute probabilities using normally distributed data

# pnorm(); the function pnorm returns the value of the cumulative density function (cdf) of the normal distribution given a certain random variable q, a population mean μ and population standard deviation σ.
# The syntax for using pnorm is as follows:
# pnorm(q, mean, sd)

## Example

q = 1.645
mean = 0
sd = 1

p = pnorm(q, mean, sd) 

### Print the outcome

print(p)
## [1] 0.9500151
